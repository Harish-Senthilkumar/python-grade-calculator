# Python Grade Calculator

This is a beginner Python project I built after completing Kaggle Learn Python.

## What it does

- Stores assignment scores in a list
- Calculates the average score
- Finds the highest score
- Finds the lowest score
- Uses an if-statement to check whether the student is passing
- Uses a function to organize the calculation

## Skills practiced

- Python lists
- Functions
- Loops
- Conditional statements
- Basic problem solving

## How to run

Run the Python file:

```bash
python grade_calculator.py
